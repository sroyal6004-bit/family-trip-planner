# 🧳 가족 여행 플래너

가족 여행 **일정**과 **경비**를 함께 기록하고 **자동 정산**하는 앱 (PWA).

## 기능
- **📅 일정** — Day 1, Day 2… 날짜별 타임라인. **장소**를 적으면 🗺️ 버튼으로 **구글지도**에서 바로 찾기
- **💰 경비** — 누가 결제했는지(개인/🏦공금), 누구랑 나눌지(N분의 1) 기록 · 카테고리별 지출 요약
- **🧮 정산** — **회비(미리 걷은 돈)** 입력 + **총무(공금 보관자)** 지정 → 멤버별 낼 돈/받을 돈 자동 계산 + 최소 송금 조합 제시 + 카톡 공유
- **☁️ 실시간 공유** — Firebase 연결 시, 가족이 **여행 코드**로 참여하면 한 사람의 수정이 **모두에게 바로 반영**
- **📁 파일 공유** — 실시간 공유 없이도 데이터를 파일로 내보내기/가져오기 가능
- 오프라인 지원, 홈 화면에 앱으로 설치 가능

## 배포 (Netlify)
GitHub 저장소를 Netlify에 연결하면 이 저장소에 **push할 때마다 자동 배포**됩니다.
- Netlify → Add new site → Import from GitHub → 이 저장소 선택 → Deploy
- 빌드 설정 없음(정적 사이트). `netlify.toml`에 이미 설정돼 있어요.

## 실시간 공유 켜기 (선택)
`firebase-config.js` 파일에 본인의 Firebase 설정값을 넣으면 실시간 공유가 자동으로 켜집니다.
1. https://console.firebase.google.com → 프로젝트 만들기
2. **빌드 > Firestore Database** 만들기 (테스트 모드로 시작)
3. **프로젝트 설정(⚙️) > 일반 > 내 앱**에서 웹앱(</>) 추가 → `firebaseConfig` 값 복사
4. 복사한 값을 `firebase-config.js`에 붙여넣고 저장 → 자동 배포되면 실시간 공유 활성화

값을 비워두면 이 기기에만 저장되는 오프라인 모드로 동작합니다.

## 사용 방법
1. 앱을 열고 **새 여행 만들기** → 이름·날짜·가족 멤버 입력
2. **일정/경비** 탭에서 + 버튼으로 기록
3. **정산** 탭에서 회비·총무를 설정하면 누가 누구에게 보낼지 자동 계산
4. 실시간 공유가 켜져 있으면 **설정 > 실시간 공유**의 코드를 가족에게 알려주기
