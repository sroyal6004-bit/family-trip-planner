# 🧳 가족 여행 플래너

가족 여행 **일정**과 **경비**를 함께 기록하고 **자동 정산**하는 앱 (PWA).

## 기능
- **📅 일정** — Day 1, Day 2… 날짜별 타임라인 (시간·분류·메모)
- **💰 경비** — 누가 결제했는지, 누구랑 나눌지(N분의 1) 기록 · 카테고리별 지출 요약
- **🧮 정산** — 멤버별 낸 돈/부담 몫 자동 계산 + 최소 송금 조합 제시 + 카톡 공유
- **📤 공유** — 여행 데이터를 파일로 내보내기/가져오기 → 가족 모두 같은 내용 보기
- 서버 없이 동작 (기기에 저장), 오프라인 지원, 홈 화면에 앱으로 설치 가능

## 사용 방법
1. 이 저장소의 **Settings → Pages → Branch: main** 선택 후 저장
2. 잠시 후 `https://<계정이름>.github.io/family-trip-planner/` 주소로 접속
3. 휴대폰 브라우저에서 열고 **"홈 화면에 추가"** 하면 앱처럼 사용
4. 가족에게도 주소를 공유하고, 여행 데이터는 ⚙️ 설정 탭의 내보내기/가져오기로 주고받기
